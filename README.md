## 古诗词乐享 微信小程序
look
### 小程序整体框架如下：
##### 数据持久化：基于微信云服务器免费版
##### 数据来源：互联网站上爬取古诗词相关数据，并同步至微信云服务器
##### 步数功能：读取微信步数，进行微信用户排行
##### 广告功能：微信小程序广告插件
